Tesla KAS44 KiCad library
Recently I have been using Tesla KAS44 Schottky diode ring for my experiments with double sideband modulation. I find this device really neat as it saves me the work of sorting out diodes to find the best matching set of 4 diodes. While I don't plan on making a PCB with them I decided to make a library for the KAS44 in KiCad.
